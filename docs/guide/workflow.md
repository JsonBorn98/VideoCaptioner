# 工作流程

了解 VideoCaptioner 的完整工作流程。

## 处理流程图

```
视频输入 → 语音识别 → 字幕分割 → [占位符清理] → 字幕优化 → 字幕翻译
        → [文本规范化] → [压缩重译] → [闪烁修复] → [阅读速度审计 / QA 报告] → 视频合成
```

方括号内为可选的规则型后处理与审计步骤，默认关闭，可在设置页 / CLI 开启，
详见 [配置指南](/guide/configuration#字幕后处理-subtitle)。
