# LLM 配置入口

当前版本把 LLM 配置拆成两部分：

- [LLM 模型方案](/config/llm)：Transport、Base URL、API Key、模型、上下文预算与并发。
- [翻译模式与双角色校对](/config/translator)：非 LLM、单 LLM、增强型双角色 LLM，
  以及术语表与审计策略。

项目不提供或推广 API 中转服务。请根据可访问性、价格、隐私和服务条款，自行选择
官方 API、本地服务或可信的兼容服务。

如果你从旧版本升级，原有 LLM 配置会迁移为命名模型方案，并在增强模式中暂时同时
绑定给主翻译与高级校对；迁移不会复制或上传 API Key。
